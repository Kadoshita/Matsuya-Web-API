# Matsuya-Web-API

松屋のメニューをランダムで返すAPIとメニューのデータ(栄養値)を返すAPI

## How to use

1. `git clone https://github.com/mak0tia/Matsuya-Web-API.git` でクローンする
1. `cd Matsuya-Web-API && npm install` をする
1. `npm start` で起動する
1. `localhost:3000/random` でランダムに松屋のAPIを返す
